Installation (quick)

  1. Download the ZIP and extract to your machine.
  2. Install Python 3.10+ and install libraries needed.
  3. Open seeds.txt, add your URLs you want to extract from.
  4. Run: python3 main.py
Note: Please respect websites that do not wish to be extracted.
